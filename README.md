# Ender 3 Pro Klipper Configuration

## Initial Setup

Ensure you have a `probe_offset.cfg` configuration file with
the following contents:

```cfg
[bltouch]
z_offset: <your Z-offset value here>
# i.e. z_offset: 0.300
```
